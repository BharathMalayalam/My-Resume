Applock

AppLock is a simple yet effective desktop application developed in Python to restrict unauthorized access to specific programs or files on a system. The primary objective of this project is to provide users with an additional layer of security for their private or sensitive applications by requiring authentication before access is granted. The project utilizes Python's tkinter library for building the graphical user interface, and os and subprocess modules for managing system-level operations. On launching AppLock, users are greeted with a login screen that prompts for a password. Once authenticated, users can add or remove applications they want to lock. The list of locked applications is stored in a secure configuration file or database, and the system constantly monitors running processes. If a locked application is detected running without proper authorization, it is immediately closed or access is denied.
